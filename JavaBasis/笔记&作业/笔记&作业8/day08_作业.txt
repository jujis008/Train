使用单例设计模式，设计一个阿里巴巴董事局主席类，然后，测试。
  要求类的对象有一些属性（名字，性别、身高、薪水）
  模拟一个场景：阿里巴巴主管外部事务的高管给来访的外宾介绍本公司首脑的情形。
【单例类中可以有 其他非私有属性，给这些属性赋值时可以用实例加点语法   仿照下面的例子写这个作业】
例如：
class Emporor
{
	String name;
	// 1. 声明一个本类的静态私有属性
	private static Emporor emporor = new Emporor();

	// 2. 私有化构造方法
	private Emporor(){}

	// 3.对外暴露一个方法 获得实例
	public static Emporor getEmporor(){
	
		return emporor;
	}
	void sleep(){
		System.out.println("睡觉");
	}
}
class  HungrySingleton
{
	public static void main(String[] args) 
	{
		Emporor e = Emporor.getEmporor();
		e.name = "雍正";
		e.sleep();
	}
}
