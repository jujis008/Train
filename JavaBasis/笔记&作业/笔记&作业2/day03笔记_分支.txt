	程序流程控制 结构化程序有三种结构
	1.顺序结构: 从上向下 一行行代码执行
	2.分支结构(选择结构)
	3.循环结构

二,分支语句
	为什么要使用分支?
		当程序面临选择的时候,每一种选择对应一种操作(结果)

	1. if语句 
		1.1 如果....就...
			结构: if(表达式) -->关系表达式 和 逻辑表达式组成
			      {
				 执行语句;
			      }
			执行流程:
				a.当程序执行到if的时候 先去判断表达式的结果是 true 还是  false
				b.如果表达式的值 是 true 就执行{}里面的内容,然后一条if结束
				c.如果表达式的值 是 false 就不会执行{}里面的内容

		1.2 如果....就...否则....

			结构:if(表达式) -->关系表达式 和 逻辑表达式组成
			      {
				 执行语句;
			      }else{
				 执行语句;
			      }
			      注意:else后面没有比较表达式 只有if后面有
		// if(){}else{} 和 三元运算符的转换问题
		//错误 运算符操作完毕 就应该有一个结果 而不是一条输出语句
		//(a>b) ? System.out.println("最大值是:"+a) : System.out.println("最大值是:"+b);
		//所有的三元运算符 都能够转成if-else语句 反之不成立
