要求 包含页面
	1.登录页面 login.html
		
		要求 html要有背景图片 
		登录框居中显示在html页面中 背景蓝色圆角透明效果

			不匹配 则在用户名/密码 文本框后输出提示信息
			
		点击注册:新开页面 -->register.html
	2.注册页面 register.html
		用户名
		密码
		确认密码
		邮箱
		
		要求:css样式 html页面左上方显示logo 右上方显示当前时间 2017年8月19日 下午 06:01:03 
		       注册框居中显示在html中 
		       js验证:用户名 密码 确认密码 不能为空 
			用户名 以字母开头 长度10位 可以包含下划线和数字 不可以包含特殊符号 例如 ! @ #...
			密码/确认密码 必须相同 密码由数字和字母组成 数字开头 字母必须包含大小写 例如 3345Qfjava
			获取表单数据 点击提交按钮 将数据存储在用户数组中(建议使用外链js) 弹出"恭喜注册成功!"提示 点击确认 跳回登录页面

			tab切换效果 显示 唐诗/宋词/元曲 切换 根据不同的tab 显示不同内容
